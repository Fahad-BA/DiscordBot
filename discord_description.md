# 🤖 Simon's Bot

✨ **A Premium Discord Music Bot** ✨

Simon's Bot isn't just another music bot — it's your server's dedicated DJ, delivering high-quality audio with a clean, reliable experience.

### 🚀 Key Features

- **Native Direct Streaming** — Powered by `yt-dlp` and `FFmpeg` for near-instant playback with zero disk overhead. No Lavalink required.
- **Full Queue Control** — Play, pause, resume, skip, stop, shuffle, loop, and manage your queue with ease.
- **Smart Idle Detection** — Automatically disconnects after 5 minutes of inactivity to save resources.
- **Volume Control** — Adjust playback volume per server (0–100%).
- **Slash Commands** — Modern `/` command interface for a seamless user experience.
