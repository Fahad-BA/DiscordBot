# 🤖 Simon's Bot | سايمون بوت

✨ **The Premium Saudi Discord Experience** ✨

سايمون بوت مهوب مجرد بوت عادي، هو رفيقك في الديسكورد اللي يفهمك ويطربك بلهجتنا النجدية الأصلية.

### 🚀 Key Features | ليش تختار سايمون؟

*   **Najdi Localization | لهجة نجدية قح**
    سايمون يسولف معك بلهجة أهل نجد (القصيم، الرياض، وكل ديرة) ومصمم خصيصاً للمجتمع السعودي.
*   **Pure Streaming | بث صافي ومستقر**
    نستخدم أحدث التقنيات `yt-dlp` و `FFmpeg` عشان نضمن لك جودة صوت خرافية وبدون أي تقطيع.
*   **Systemd Performance | أداء جبار واستمرارية**
    البوت شغال بنظام `systemd` لضمان استقرار عالي وأداء سريع على مدار الساعة.
